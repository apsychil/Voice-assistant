# Asistente personal
Este pequeño programa es un asistente personal al cual le puedes pedir diversas cosas a través del micrófono. Entre las cosas que puede hacer están: buscar en google, buscar en wikipedia, reproducir videos de youtube, dar la hora y la fecha, buscar acciones en Yahoo Finance, contar chistes, abrir diversos sitios web directamente, entre otros.

Se ha utilizado para este proyecto las siguientes librerias: pyttsx3, speech_recognition, pywhatkit, yfinance, pyjokes, webbrowser, datetime y wikipedia.
